# ClackHouse Website

Static marketing site and documentation for ClackHouse.

## Stack

- Astro
- Starlight
- Tailwind CSS v4

## Commands

```bash
pnpm install
pnpm dev
pnpm build
pnpm check
```

Or with `just`:

```bash
just install
just dev
just build
just check
```

## Structure

- `src/pages/` for the marketing site
- `src/content/docs/` for Starlight docs content
- `src/styles/` for the site and docs theme
